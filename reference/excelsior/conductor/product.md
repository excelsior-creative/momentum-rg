# Product Guide

## Initial Concept

The purpose of this project is to establish a public-facing marketing website for a digital agency called Excelsior Creative. It is built using Payload CMS and Next.js and is a combination of static website pages and SEO-optimized articles and work examples that serve as case studies for our work. The goal of this website is to be very fast and performant so that it loads quickly for visitors and provides a very positive image of our agency.
